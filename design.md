### 文本颜色
- TextFillColorPrimary : 主要文本颜色
- TextFillColorSecondary : 次要文本颜色（如描述性文本）
- TextFillColorTertiary : 第三级文本颜色（如提示性文本）
- TextFillColorDisabled : 禁用状态的文本颜色
- TextFillColorInverse : 反色文本（在深色背景上的文本）
### 控件基础颜色
- ControlFillColorDefault : 控件默认背景色
- ControlFillColorSecondary : 控件次要背景色（如悬停状态）
- ControlFillColorTertiary : 控件第三级背景色（如按下状态）
- ControlFillColorDisabled : 控件禁用状态背景色
- ControlFillColorInputActive : 输入控件激活状态背景色
### 边框颜色
- ControlStrokeColorDefault : 控件默认边框颜色
- ControlStrokeColorSecondary : 控件次要边框颜色
- ControlStrokeColorOnAccentDefault : 强调色上的边框颜色
- FocusStrokeColorOuter : 焦点外边框颜色
- FocusStrokeColorInner : 焦点内边框颜色
### 卡片/面板颜色
- CardBackgroundFillColorDefault : 卡片默认背景色
- CardBackgroundFillColorSecondary : 卡片次要背景色
- CardStrokeColorDefault : 卡片边框颜色
- CardStrokeColorDefaultSolid : 卡片实心边框颜色
### 层次颜色
- LayerFillColorDefault : 默认层叠背景色
- LayerFillColorAlt : 替代层叠背景色
- LayerOnAcrylicFillColorDefault : 亚克力效果上的层叠颜色
- LayerOnMicaBaseAltFillColorDefault : Mica效果上的层叠颜色
### 实体背景颜色
- SolidBackgroundFillColorBase : 基础实体背景色（如窗口背景）
- SolidBackgroundFillColorSecondary : 次要实体背景色
- SolidBackgroundFillColorTertiary : 第三级实体背景色
- SolidBackgroundFillColorQuarternary : 第四级实体背景色
### 系统状态颜色
- SystemFillColorSuccess : 成功状态颜色
- SystemFillColorCaution : 警告状态颜色
- SystemFillColorCritical : 错误状态颜色
- SystemFillColorNeutral : 中性状态颜色
### 分隔线颜色
- DividerStrokeColorDefault : 默认分隔线颜色
### 特殊控件颜色
- ControlOnImageFillColorDefault : 图片上控件的默认背景色
- ControlOnImageFillColorSecondary : 图片上控件的次要背景色
- ControlOnImageFillColorDisabled : 图片上控件的禁用状态背景色
### 强调色相关
- AccentFillColorDisabled : 强调色禁用状态
- TextOnAccentFillColorPrimary : 强调色上的主要文本颜色
- TextOnAccentFillColorSecondary : 强调色上的次要文本颜色
- TextOnAccentFillColorDisabled : 强调色上的禁用文本颜色
### 表面/遮罩颜色
- SmokeFillColorDefault : 默认遮罩层颜色
- SurfaceStrokeColorDefault : 表面边框默认颜色
- SurfaceStrokeColorFlyout : 弹出层边框颜色

### 控件替代颜色
- ControlAltFillColorSecondary : 控件替代背景色（次要）
- ControlAltFillColorTertiary : 控件替代背景色（第三级）
- ControlAltFillColorQuarternary : 控件替代背景色（第四级）
- ControlAltFillColorTransparent : 控件替代背景色（透明）
- ControlAltFillColorDisabled : 控件替代背景色（禁用）
### 微妙填充色
- SubtleFillColorTransparent : 微妙填充色（透明）
- SubtleFillColorSecondary : 微妙填充色（次要）
- SubtleFillColorTertiary : 微妙填充色（第三级）
- SubtleFillColorDisabled : 微妙填充色（禁用）
  主要用于需要轻微视觉区分的场景
### 强调控件相关
- ControlStrokeColorOnAccentTertiary : 强调色上的第三级边框颜色
- ControlStrongFillColorDefault : 强调控件填充色（默认）
- ControlStrongFillColorDisabled : 强调控件填充色（禁用）
- ControlStrongStrokeColorDefault : 强调控件边框色（默认）
- ControlStrongStrokeColorDisabled : 强调控件边框色（禁用）
### Mica 效果相关
- LayerOnMicaBaseAltFillColorSecondary : Mica效果上的次要层叠颜色
- LayerOnMicaBaseAltFillColorTertiary : Mica效果上的第三级层叠颜色
- LayerOnMicaBaseAltFillColorTransparent : Mica效果上的透明层叠颜色
### 系统状态背景色
- SystemFillColorAttentionBackground : 注意状态背景色
- SystemFillColorSuccessBackground : 成功状态背景色
- SystemFillColorCautionBackground : 警告状态背景色
- SystemFillColorCriticalBackground : 错误状态背景色
- SystemFillColorNeutralBackground : 中性状态背景色
- SystemFillColorSolidAttentionBackground : 实心注意状态背景色
- SystemFillColorSolidNeutralBackground : 实心中性状态背景色
### 特殊场景颜色
- ControlStrokeColorForStrongFillWhenOnImage : 图片上强调填充的边框颜色
- SurfaceStrokeColorInverse : 反色表面边框颜色
- LayerOnAccentAcrylicFillColorDefault : 强调色上的亚克力效果填充色
